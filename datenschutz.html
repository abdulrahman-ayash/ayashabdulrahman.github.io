2. Geltungsbereich

Diese Datenschutzerklärung gilt für die Nutzung der App „PDF Syn“ (im Folgenden „App“) auf Ihrem Endgerät (Smartphone, Tablet). Sie gilt nicht für Websites Dritter, auf die in der App verlinkt wird.

3. Verarbeitete Daten und Zwecke

3.1 Grundsatz: Lokale Verarbeitung
Die App verarbeitet Ihre PDF‑Dateien, Scans und Einstellungen standardmäßig ausschließlich lokal auf Ihrem Gerät. Es findet keine automatische Übermittlung personenbezogener Daten an uns oder Dritte statt.

3.2 Kamera (Dokumentenscanner)
Für die Scanner-Funktion greift die App auf die Kamera Ihres Geräts zu. Die dabei entstehenden Bilder werden nur lokal verarbeitet und nicht an uns oder Dritte übermittelt. Eine Speicherung erfolgt nur, wenn Sie dies aktiv speichern.

3.3 OCR-Funktion (Texterkennung)
Die Texterkennung (OCR) erfolgt vollständig lokal auf Ihrem Gerät. Es werden keine Bild- oder Textdaten an externe Server gesendet.

3.4 Zahlungs-QR-Codes
Bei der Erstellung von QR-Codes für Zahlungen (IBAN, Empfänger) geben Sie die Daten selbst ein. Diese Daten werden nicht von uns gespeichert oder übermittelt. Der generierte QR‑Code enthält die von Ihnen eingegebenen Informationen und wird lokal auf Ihrem Gerät erstellt.

3.5 Nutzung von Firebase (optional)
Um Ihnen erweiterte Funktionen wie eine verschlüsselte Synchronisation oder eine Benutzeranmeldung anbieten zu können, nutzen wir Firebase, einen Dienst von Google LLC. Firebase wird nur dann aktiv, wenn Sie sich freiwillig für ein Benutzerkonto entscheiden. In diesem Fall werden folgende Daten an Firebase übermittelt und dort gemäß der Firebase‑Datenschutzerklärung verarbeitet:

  • E‑Mail‑Adresse und Authentifizierungsdaten (bei Registrierung)
  • Geräteinformationen (z. B. Betriebssystemversion, Gerätetyp) zur Fehleranalyse
  • ggf. von Ihnen erstellte Dokumente, wenn Sie die Cloud‑Sicherung aktivieren (diese werden dann Ende‑zu‑Ende verschlüsselt)

Die Nutzung von Firebase erfolgt auf Grundlage von Art. 6 Abs. 1 lit. a DSGVO (Einwilligung) oder lit. f (berechtigtes Interesse an einer sicheren und komfortablen Benutzerverwaltung). Sie können die Nutzung jederzeit in den App‑Einstellungen deaktivieren, was jedoch zum Verlust der Cloud‑Funktionen führen kann.

3.6 Keine Weitergabe an sonstige Dritte
Eine Übermittlung Ihrer personenbezogenen Daten an andere Dritte erfolgt nicht, es sei denn, Sie nutzen explizit eine Funktion, die eine solche Weitergabe erfordert (z. B. Teilen einer Datei über das Betriebssystem). In diesen Fällen entscheiden Sie selbst über die Datenweitergabe.
